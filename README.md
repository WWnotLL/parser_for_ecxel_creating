Программа предназначена для перебора данных из директории и формирования необходимого Excel-файла.
Важно, что бы указанный файл лежал в корне той же директории, что и файлы, которые необходимо обработать. Например, если у вас есть директория dir_folder, в которой есть папки, в которых лежат нужные файлы, чоответствующие указанному шаблону, 
структура должны выглядеть следующим образом:
|-dir_folder
|---folder_1
|------file_1
|------file_2
|---folder_2
|---folder_3
|---...
|---folder_n
|---Parser.py
